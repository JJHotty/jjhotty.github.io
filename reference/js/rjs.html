Flexbox

Flexbox is a CSS Web layout model used for responsive Web design. The tool allows a developer to manipulate the page by sizing and structuring the various parent, and children div classes, id's and html classes around.
Flexbox offers a really flexible way to move items around that will responsively grow and shrink dependant on the screen size, and the size of the parent.
